プロジェクト

mogitate

ECサイトをテーマにしたスクール課題で、商品の一覧表示や詳細管理機能などを備えたWebアプリケーションを開発しました。

LaravelのMVC構造やルーティング設計を意識し、短期間での高品質な実装を目指しました。検索・並び替え・ページネーションなどの機能を組み込んで、実用性の高いUIを実現しました。

使用技術：PHP / Laravel / MySQL / Tailwind CSS

開発期間：1週間

評価：S判定（スクールより優秀作品として高評価）

主な機能：

  • 商品一覧表示（6件ずつのページネーション）
  • 商品名検索（部分一致対応）
  • 価格による並び替え（高い順・安い順）
  • モーダルによる並び替え条件の表示・解除
  • 商品詳細ページでの編集・削除・バリデーション対応
  • 画像アップロード（ローカル対応）

フリマアプリ

実際のサービスを想定し、出品・購入・プロフィール編集・レビューなどの機能を備えたフリマアプリを開発しました。

ユーザー認証やRESTful API、Stripeによる決済、Mailtrapによるメール認証機能を通して、フルスタックの開発スキルを強化しました。UI/UXも直感的に使えるよう意識しました。

使用技術：PHP / Laravel / MySQL / Tailwind CSS / Stripe / Mailtrap / Fortify / FormRequest

開発期間：2ヶ月（うち1ヶ月で完成）

評価：S判定（高い完成度を評価される）

主な機能：

  • ユーザー登録／ログイン（Fortify + メール認証）
  • 商品一覧・詳細・検索（商品名検索、カテゴリ・状態の表示）
  • 商品出品・画像アップロード（複数カテゴリ・商品状態選択）
  • 「いいね」機能、コメント投稿・表示
  • 商品購入（Stripe決済／支払方法選択／配送先変更）
  • プロフィール編集（画像、ユーザー名、住所変更）

勤怠管理アプリ

打刻管理、修正申請、承認など実務を想定した勤怠管理システムを開発しました。

Fortifyによる認証、FormRequestによるバリデーション、管理者と一般ユーザーの権限分離を設計し、堅牢かつ実用的なWebアプリを構築しました。

使用技術：PHP / Laravel / MySQL / Tailwind CSS / Mailtrap / Fortify / FormRequest

開発期間：約3週間（予定より大幅に前倒し）

評価：S判定（スクール評価で高評価）

主な機能：

  • ユーザー登録／ログイン／メール認証（Mailtrap）
  • 出勤／退勤／休憩（複数対応）
  • 勤怠一覧（月別／日別）、打刻ステータス管理
  • 修正申請・承認機能（ユーザー ⇄ 管理者）
  • CSV出力機能、UI制御（権限ごと）
  • 詳細なバリデーションとエラーメッセージ表示
